Kotlin кроссплатформенный статически типизированный, высокоуровневый, язык программирования с выводом типов. Расчитан на полное взаимодействие с Java. Стандартная библиотека Kotlin в JVM-версии зависит от библиотеки классов Java, но вывод типов позволяет сделать синтаксис более лаконичным. Компилируется как в JavaScript, так и в нативный байткод LLVM (low level virtual machine).

7 мая 2019 компания Google объявила, что Kotlin — предпочитаемый язык для разработки Android приложений. С момента выхода Android Studio версии 3.0 в октябре 2019. Kotlin был включен в качестве альтернативы компилятора Java. По умолчанию исходный код Kotlin компилируется в байткод Java.

История

Название происходит от острова российского острова Котлин в Финском заливе, рядом с Санкт-Петербургом, как название языка программирования Java происходит от индонезийского острова Ява.
